# 🎙️ VoiceCalc Pro

A modern voice-enabled calculator built using HTML, CSS, and JavaScript. VoiceCalc Pro allows users to perform calculations manually or through voice commands while offering features such as dark/light mode, persistent history storage, PDF export, and responsive design.

---

## 🚀 Features

### 🧮 Calculator Operations
- Addition (+)
- Subtraction (-)
- Multiplication (×)
- Division (÷)
- Percentage (%)

### 🎤 Voice Control
- Perform calculations using voice commands
- Supports commands like:
  - "5 plus 3"
  - "10 minus 4"
  - "8 times 7"
  - "20 divided by 5"
- Voice output announces the result

### 🌙 Theme Support
- Dark Mode
- Light Mode
- Theme preference saved automatically

### 📄 PDF Export
- Export calculation history as a PDF document
- One-click download

### 💾 Local Storage
- Saves calculation history
- Stores theme preference
- Tracks total calculations

### 📱 Responsive Design
- Optimized for:
  - Mobile Phones
  - Tablets
  - Laptops
  - Desktop Computers

### ⌨️ Keyboard Shortcuts
| Key | Action |
|------|---------|
| Enter | Calculate |
| Backspace | Delete Last Character |
| Escape | Clear Display |
| 0-9 | Number Input |
| + - * / % | Operators |

## 🎯 How to Use

### Manual Calculation
1. Click number buttons.
2. Select an operator.
3. Press `=` to calculate.

### Voice Calculation
1. Click **🎤 Voice Input**.
2. Speak a mathematical expression.
3. The application calculates and announces the result.

### Export History
1. Click **📄 Export PDF**.
2. Download the generated PDF containing calculation history.

### Change Theme
1. Click the theme toggle button in the top-right corner.
2. Switch between Dark and Light modes.
